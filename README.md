Fortune Wheel - Customisable with Preset
===
A modern, responsive, fully client‑side **spinning wheel app** built for https://fortune.gunjanpatel.info


## 🚀 Features

### 🎯 Wheel
- Dynamic wedge count  
- Automatic color generation  
- Smooth CSS rotation  
- Works on desktop & mobile  
- High‑resolution pointer  
- Realistic “tick” sound as wheel spins
